# nurse-x-lp

看護師転職noteから誘導するための公開LPです。

## 公開URL

GitHub Pagesを有効化すると、以下のURLで公開されます。

```text
https://hidechika3.github.io/nurse-x-lp/
```

## GitHub Pages設定

新しいPublicリポジトリ `nurse-x-lp` を作成後、以下を設定します。

```text
Settings → Pages
Source: Deploy from a branch
Branch: main
Folder: /root
```
